# Assistente-Am-lia
Estou desenvolvendo a **Assistente Amélia**, um projeto de Inteligência Artificial criado para interagir com usuários, responder perguntas e executar tarefas de forma inteligente. O sistema está sendo construído com foco em backend, APIs e arquitetura de software. Ainda em evolução, a Amélia faz parte da minha jornada no desenvolvimento IA.
